CGI bedrijfsonderzoek

Namen: Ahmet Demir & Sebastiaan van Etten
Klas: INF1B
Inleiding
Wij als klas INF1B hebben de opdracht gekregen om vragen op te stellen voor het bedrijfsuitje. Sebastiaan en ik hebben besloten om samen te werken en hebben gebrainstormed voor vragen. We zijn afgelopen woensdag vertrokken om 9 uur met de metro naar Alexandrium. We hebben een presentatie gekregen over wat CGI is en daarna een rondleiding gekregen door de aantal projecten die CGI heeft afgerond.

Wat is CGI?
- Conseillers engestion et informatique (afgekort CGI), is een groot multi-national IT bedrijf die op het vijfde plaats zit op de ranglijst van de grootste IT bedrijven op de wereld.
- Het is Frans-Canadees bedrijf die een annual revenue heeft van 10 miljard euro met ongeveer 400 bedrijven in 40 locaties in de wereld. CGI heeft 68000 medewerkers.
- Zij richten zich het meest op het klant, want ze willen zoveel mogelijk miscommunicatie voorkomen. De miscommunicatie kan er voor zorgen dat de doorlooptijd van een project te lang zal duren wat ook te veel geld zou kosten.
- Door de communicatie hebben zij een klant tevredenheids cijfer: 9, want 95% van de projecten van CGI worden op tijd en binnen budget afgerond.
- Ze houden equilibrium tussen de klanten, de stakeholders en de medewerkers, zodat iedereen op zijn/haar gemak kan werken en projecten op tijd worden afgerond. Dit zorgt ervoor dat er prettige werkomstandigheden voor de medewerkers komen.
- Een paar voorbeelden van welke sectoren CGI invloed heeft zijn: Zorg, Overheid, Transport, Communicatie en Post en Logistiek.

Het project dat wij gaan belichten is Burgernet.
- Het project was eerst ontstaan uit opdracht van een aantal gemeenten. Na groot succes is dit uitgebreid tot op landelijk niveau. Burgernet is een product waaraan CGI meewerkt. CGI is verantwoordelijk voor het beheren, onderhouden en eventuele ontwikkelingen van de applicatie. De ICT-infrastructuur en de levering van de SMS- en telefoondiensten zijn ook onderdeel van CGI's werk aan de applicatie, dit word geregeld via een externe partij.
- De gebruikers van de applicatie zijn de burgers die zich hiervoor hebben aangemeld. De bedoeling is dat de mensen op de hoogte worden gesteld wanneer er een ongeval heeft plaats gevonden of wanneer er naar iets of iemand wordt gezocht.
- CGI is een bedrijf dat erg de nadruk legt op de klant door middel van communicatie. De projectmethodiek die ze hebben gehanteerd is er een die erg veel lijkt op scrum, Ze communiceren zo veel mogelijk met de klant om calamiteiten te voorkomen.
- Over de werking en opbouw van de applicatie is niet veel vermeld, wel is verteld dat de manier waarop het product werkt en in welke programmeertaal het is geschreven vaak afhangt voor welke klant ze het maken. Er werdt niet verteld over de mensen die er aan gewerkt hebben en de verschillende rollen, er werd simpelweg verwezen naar werknemers . Er is niet specifiek verteld wie wat deed en wat voor rol zij speelde in het proces van het creeren van burgernet.

Opdracht: De opdracht was om een programma te schrijven die een wachtwoord kan beoordelen. Je programma moest zeggen of iemands wachtwoord slecht, redelijk of goed is.
Taal: python
Uitleg: Ik heb ervoor gekozen om een programma te schijven die één voor één de letters doorgaat van het ingevoerde woord. Elke letter wordt dan beoordeeld op wat het is. Uiteindelijk worden deze beoordelingen bij elkaar opgeteld en komt er een score uit.
x = str(input("voer wachtwoord in:"))
count = 0
countlower = 0
counthigher = 0
countgetal = 0
countspace = 0
countspec = 0
cijfers = "0123456789"
special = ",.!@#~$%^&*()-_=+[{]};:'<>?"
while count < len(x):
    y = x[count]
    if y in cijfers:
        countgetal += 1
    elif y in special:
        countspec += 1
    elif y == " ":
        countspace += 1
    elif y == y.lower():
        countlower += 1
    elif y == y.upper():
        counthigher += 1
    count += 1
score = countlower + counthigher*1.7 + countgetal*1.9 + countspace*2.6 + countspec*3
if score < 10:
    print("slecht")
elif score < 30:
    print("medium")
elif score > 30:
    print("noice!")
print("je score is:",("%.2f") % score)

Opdracht: Dit is de eerste opdracht waarbij het opeens een stukje moeilijker werd dan de vorige. Bij deze opdracht moesten we een programma maken die een bericht kon encrypten. Het was de bedoelingen dan alle letters, hoofdletter of klein, met 5 werden verzet. Alle getallen en speciale tekens moesten onaangetast blijven. En om het af te maken mocht je maar één keer 'print()' gebruiken in je code.
Taal: python
Uitleg: Bij deze opdracht was het even nadenken hoe je het voor elkaar gaat krijgen, want je moet rekening houden met de acscii tabel. Je moet eerst weten op welke plek je letter staat in de acscii tabel en dan kan je het vijf stappen verschuiven. Het probleem waar ik eerst tegenop liep was dat ik de letters over hun bereik liet gaan, dus soms werden kleine letters hoofdletters en andersom ook. Erger nog, soms kwamen er uit het niets onbekende tekens tussen staan. Dat had ik weten te voorkomen door een bereik te geven aan de letters. Alle hoofdletters bleven vast tussen de 65-90 en de kleine letters bleven tussen de 97-122. Hieronder staat de uiteindelijke code die ik ingeleverd heb.
x = input("voer wat in:")
y = int(input("verplaats met hoeveelheid:")) %26
output = ""
lijst = "!@#$%^&*()_-+=]}[{';:,./?><0123456789 "
for i in x:
    a = ord(i) + y
    if i in lijst:
        output += i
    else:
        if ord(i) >= 65 and ord(i) <=90:
            if a > 90:
                a = a - 26
            elif a < 65:
                a = a + 26
        elif ord(i) >= 97 and ord(i) <= 122:
            if a > 122:
                a = a - 26
            elif a < 97:
                a = a + 26
        b = chr(a)
        output += b
print(output)



Opdracht: Bij deze opdracht was het de bedoeling dat je een rondje maakt van tekens. Er waren bepaalde regels waaraan je code moest voldoen: je mag geen string met een getal vermenigvuldigen, je code mag maar één 'print()' bevatten, er MOET minstens één loop in je code, het rondje moet schaalbaar zijn op verschillende afmetingen en je mag alleen gebruik maken van 'math.sqrt' om het rondje te maken.
Taal: python
Uitleg: Bij deze code creëer ik eerst een raster. Vervolgens bereken ik een rondje, alles wat binnen dat rondje valt krijgt een '#'. De rest wordt opgevuld met een spatie.
import math
inp = int(input("grootte:"))
output = ""
straal = inp/2
for y in range(inp):
    for x in range(inp):
        distance = math.sqrt(((straal - x)**2) + ((straal - y)**2))
        if distance < straal:
            output += "#"
        else:
            output += " "
    output += "\n"
print(output)
